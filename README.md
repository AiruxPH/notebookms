# notebook
CSIT601 preliminary exam requirement

## Requirements (for reference)
### 1.  Text storage
  For a note app to be considered a note app it requires that the user have the ability to write text. This text will be stored on the app for later reading by the user.
### 2.  Sorting and Filtering
  The user may have the ability to sort and filter through their texts in order to find a specific set of texts they may be in search of.
### 3.  Categories
  Gives the user the ability to organize their notes according to category. They may be color-coded to make the notes easier to find.
### 4.  Search
  This feature gives the user the ability to search notes accordingly.
### 5.  Note statistics
  Shows word count, character count, and file size. This is useful for when the user needs to keep track of the size of their text.
### 6.  Archiving
  This gives the user the ability to move old or completed notes without completely removing it.
### 7.  Pinning
  The user may pin important notes so that they show up at the top of the list.
### 8.  Dashboard
  Acts as the landing page after the user has logged in, showing ads and announcements by the developer. Along with showing pinned notes and a button to the normal display page, this page also shows a list of categories that the user may quick-search/quick-sort with.
### 9.  Date/Time stamp
  This feature is able to record the creation date and the last modified date/time for each note provides essential context.
### 10.  Basic Rich Text Formatting
  This allow the users to apply basic text formatting makes notes more readable and allows the user to bring more emphasis to certain parts.
